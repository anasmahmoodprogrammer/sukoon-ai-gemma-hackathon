# ЁЯУЦ Overview

Mental health support is often difficult to access because of language barriers, social stigma, limited resources, and high costs.

**Sukoon AI** bridges this gap by combining the reasoning capabilities of **Google Gemma 4** with a simple, soothing user experience. Users can communicate naturally in **English, Urdu, or Hindi** through text or voice while receiving compassionate guidance tailored to their cultural preferences.

The application is designed to prioritize **privacy**, **accessibility**, and **user safety**.

# ЁЯЪА Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/your-username/sukoon-ai.git

cd sukoon-ai
```

---

## 2. Install Dependencies

### Backend

```bash
cd server

npm install
```

### Frontend

```bash
cd client

npm install
```

---

## 3. Environment Variables

Create a `.env` file inside the server directory.

```env
GEMINI_API_KEY=YOUR_API_KEY
PORT=3000
```

---

## 4. Run Backend

```bash
npm start
```

or

```bash
node server.js
```

---

## 5. Run Frontend

```bash
npm run dev
```

---

## 6. Open Application

```text
http://localhost:5173
```
